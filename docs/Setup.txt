- installare visual studio code
- installare node.js
- installare mongodb - community server:	https://www.mongodb.com/download-center?jmp=nav#community
- aggiungere la path della cartella che contiene mongod.exe (es. C:\Program Files\MongoDB\Server\4.0\bin\mongod.exe) alla variabile di ambiente "Path" (utente o sistema)
- riavviare perchè la shell "veda" la nuova variabile (provare anche riavviando solo vscode)
- npm install -g @angular/cli (mette l'ultima versione, quando viene lanciato un comango "ng" fa da solo il check della versione e, se quella globale è più recente, usa quella del workspace)
